# Kerolos Sfwat - Portfolio & Admin Panel

مرحباً بك في مستودع الكود الخاص بموقع السيرة الذاتية (Portfolio) ولوحة التحكم. هذا الموقع مبني بأحدث التقنيات ليكون سريعاً ومتجاوباً مع جميع الأجهزة.

## 🔗 روابط الموقع (Live Links)

بمجرد رفع هذا الكود على استضافة مجانية (مثل Vercel أو Netlify أو GitHub Pages)، ستتمكن من الدخول إلى الموقع عبر الروابط التالية:

1. 🌐 **الموقع الرئيسي (للعرض للجمهور):**
   `https://[your-domain]/`

2. ⚙️ **لوحة التحكم (لإضافة وتعديل المشاريع):**
   `https://[your-domain]/#/admin-panel`

*(ملاحظة: استبدل `[your-domain]` بالرابط الفعلي الذي ستحصل عليه بعد رفع الموقع).*

---

## 🚀 كيفية رفع الموقع مجاناً ليعمل على جميع الأجهزة
أفضل وأسهل طريقة لرفع هذا الموقع هي باستخدام **Vercel**:
1. قم برفع هذا الكود إلى حسابك في GitHub.
2. اذهب إلى موقع [Vercel.com](https://vercel.com) وقم بتسجيل الدخول بحساب GitHub الخاص بك.
3. اضغط على `Add New` ثم `Project`.
4. اختر هذا المستودع (Repository) واضغط `Import`.
5. اضغط `Deploy` وانتظر دقيقة واحدة.
6. مبروك! موقعك الآن يعمل على جميع الأجهزة في العالم.

---

## 💻 كيفية التشغيل محلياً على جهازك (Local Development)

إذا كنت تريد تشغيل الموقع على جهاز الكمبيوتر الخاص بك للتعديل عليه:

1. قم بتثبيت حزم النود (Dependencies):
   ```bash
   npm install
   ```

2. قم بتشغيل خادم التطوير:
   ```bash
   npm run dev
   ```

3. افتح المتصفح على الرابط: `http://localhost:3000`

---

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **Frontend:** React.js, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (Motion/React)
- **Icons:** Lucide React
- **Backend/Database:** Firebase (Firestore & Authentication)
- **Routing:** React Router DOM (HashRouter)
